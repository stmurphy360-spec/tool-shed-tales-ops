# Privacy Policy — Tool Shed Tales Ops

_Effective date: June 11, 2026_

Tool Shed Tales Ops ("the tool") is an internal command-line application operated by Green Growth Digital LLC ("we," "us"), a single-member LLC, solely to manage the YouTube channel we own, [Tool Shed Tales](https://www.youtube.com/@ToolShedTales). The tool has exactly one user: the LLC's sole member, who is also the channel owner.

## Use of YouTube API Services

The tool uses **YouTube API Services** (the YouTube Data API and the YouTube Analytics API). By authorizing the tool, the (sole) user agrees to the [YouTube Terms of Service](https://www.youtube.com/t/terms). Google's handling of data is described in the [Google Privacy Policy](http://www.google.com/policies/privacy).

## What data the tool accesses

- **Our own channel's data only:** the video list, video metadata (titles, descriptions, tags, privacy status), and channel performance reports (views, watch time, retention, subscribers, traffic sources) for the Tool Shed Tales channel, which we own.
- **OAuth credentials:** an access/refresh token authorizing the tool to act on our own channel.

The tool never requests, accesses, collects, or stores data about any other person, user, channel, or video. It serves no third parties, has no website visitors, and uses no cookies or tracking technologies.

## How data is stored and shared

- All data retrieved from the YouTube API Services is stored in **local files on the owner's personal computer** and used only for the owner's review of channel performance.
- OAuth tokens are stored in a local file on the same computer and are transmitted only to Google's OAuth endpoints.
- **No data is sold or shared with any third party.** No data is transferred to any internal or external server.
- Stored API data is refreshed or deleted at the owner's discretion; analytics reports older than needed may be deleted at any time. No API data is retained beyond the owner's local files.

## Revoking access

Access granted to the tool can be revoked at any time via Google's security settings page: [https://security.google.com/settings/security/permissions](https://security.google.com/settings/security/permissions). Revoking access immediately invalidates the stored tokens. Locally stored data can additionally be deleted by deleting the local files.
